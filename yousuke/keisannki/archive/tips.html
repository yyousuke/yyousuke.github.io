１、SuSE Linux 9.2

  • ネットワークインターフェースがない
    SuSE Linuxで起こった問題と解決策についてメモしておきます。
    このマシンはループバックインターフェースloとeth0とeth1の２つのインター フェースを持つシステムですが、リブートした際にeth0, eth1がなくなりました。
    $ ifconfig
    を試してもloしか出てきません。

    解決策： log (boot.msg)を調べた所、
    ls: /sys/class/net/: No such file or directory
    doneWaiting for mandatory devices: eth-id-(Mac Address)
    failed (Mac Address) No interface found
    とのエラーが出ていました。
    ファイルシステムを調べた所、PCIデバイスがマウントされるはずの /sys がな くなっていました。
    $ mkdir /sys としてリブートしたらPCIデバイスが正常にマウントされ、解決し ました。
