emacsでの日本語入力の設定

 

現在(2004年9月9日時点)のbetaでは日本語化されたemacsは存在するものの日本語入力はできない状態になっています。このままでは不便なので日本語入力ができるようにしてみようと思いました。

 

１、現在の設定の確認

betaにインストールされているemacsを確認したところ、/usr/local/bin/emacs-19.34のみがうまく起動したので、これを改良して日本語入力をしようと試みました。

　まず、Wnnがインストールされているかを調べ、次に日本語入力を行うクライアントwnnとwnnサーバ(jserver)が機能しているかを調べました。wnn はWnn4/ディレクトリが存在するのでVersion4がインストールされていることは分かりました。次にwnnサーバ(jserver)が機能しているかを調べました。wnnサーバ(jserver)が起動しているかは

$ /usr/i18n/bin/Wnn4/wnnstat

で調べることができます。

次のように表示されればjserverは起動しています。

------------------------------------------------------------------------------------------------------------------------

User name:Host name     (socket)        Env No.

------------------------------------------------------------------------------------------------------------------------

jserverを使っているユーザがいる場合には下の例のようになります。

------------------------------------------------------------------------------------------------------------------------

User name:Host name     (socket)        Env No.

yousuke:unix            (0)             0 1

------------------------------------------------------------------------------------------------------------------------

 

 

２、jserverの起動

今回の場合、jserverは起動していなかったので

$/usr/i18n/bin/Wnn4/jserver

として手動でjserverを起動しました。成功すると辞書を読み込んだというメッセージが出ます。

 

 

３、wnnの設定

　jserverが起動したらemacsを設定ファイルなしで起動し、wnnが使えるか調べました。

/usr/local/bin/mule -> /usr/local/bin/emacs-19.34 とリンクされているので

$mule –q

とします。

今回の場合、wnnは起動しひらがなの入力は行えましたが“ファイルのI-nodeとFILE_UNIQを一致させることができません“というエラーが出て漢字に変換することができませんでした。もう一度変換しようとすると”付属語ファイルを読み込めません“というエラーが出ました。

　本来ならインストール時にwnntouchが実行され、辞書のヘッダを書き換えて辞書のinodeと合わせるはずですが、/usr/local/lib/wnn/ja_JP/dic/full.fzk(付属語ファイル)に対してwnntouchが実行されていなかったためこのようなエラーが出たようです。

解決法として

$/usr/local/bin/Wnn4/wnntouch  /usr/local/lib/wnn/ja_JP/dic/pubdic/full.fzk

を実行し付属語ファイルが読み込めるようにしました。

 

 

４、betaのemacsの使用法

　今回はたまごでwnnを使用するという方針で設定を行いましたが、このためには各自のホームディレクトリに.emacsと.eggrcを置き、.emacsでwnnとたまごを呼び出すように指定する必要があります。このファイルは/net/beta/home/yousukeにありますので必要ならコピーして利用して下さい。ただ、artemisでemacsを移動する際にはエラーが出ますのでそのときには(load "wnn-egg")の行をコメントアウトして下さい。
