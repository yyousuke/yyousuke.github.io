アーカイブ：HDF4.2r3のインストール（Solaris）


2010年3月22日に更新した情報のアーカイブです。


  ここでは、Solaris9でHDF4.2をインストールした際の設定方法をメモしています。
  コマンド入力の説明では、%が一般ユーザの場合のプロンプト、#がrootの場合のプロンプトとして解説しています。



1　HDF4のダウンロード

  HDFのページからHDF4のソースコードとSZIP、ZLIB、JPEGをダウンロードします。


2　SZIPのインストール

  ここでは、/usr/local以下にインストールすることを想定しています。別の場所にインストールする場合、configure で --prefix=インストール先　を変更してください。
% gunzip -c szip-2.1.tar.gz | tar xvf - 
% cd szip-2.1
% ./configure --prefix=/usr/local
% make 

% make checkで
All test passed.が出ることを確認し、
# make install


3　ZLIBのインストール

  ここでは、/usr/local以下にインストールすることを想定しています。別の場所にインストールする場合、configure に --prefix=インストール先　のオプションを付けてください。
% gunzip -c zlib-1.2.3.tar.gz | tar xvf -
% cd zlib-1.2.3
% ./configure
% make
# make install


4　JPEGのインストール

  ここでは、/usr/local以下にインストールすることを想定しています。別の場所にインストールする場合、configure に --prefix=インストール先　のオプションを付けてください。
% gunzip -c jpegsrc.v6b.tar.gz | tar xvf -
% cd jpeg-6b
% ./configure
% make
# make install


5　HDF4.2r3のインストール

  ここでは、/usr/local以下にインストールすることを想定しています。別の場所にインストールする場合、configure に --prefix=インストール先　のオプションを付けてください。
% gunzip -c HDF4.2r3.tar.gz | tar xvf -
% cd HDF4.2r3

% setenv CPPFLAGS -I/usr/local/include
% ./configure --prefix=/usr/local/HDF4.2r3 --with-zlib=/usr/local/lib --with-jpeg=/usr/local/lib \
--with-szlib=/usr/local/lib 

% make

% make check でテストが成功したことを確認し、
# make install
＊ HDF4.2.5でも同じ方法でインストール可能です。
